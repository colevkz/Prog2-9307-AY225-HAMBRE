# Mini Data Analytics Console Dashboard

A simple analytics dashboard for video game sales data with both Java console and HTML web versions.

## Features

- View Dataset Summary
- Monthly Sales Analysis
- Top Publishers Ranking
- Genre Analysis

## Java Version

### Requirements
- Java JDK 8 or higher

### How to Run
1. Place vgchartz-2024.csv in the same directory as AnalyticsDashboard.java
2. Compile: `javac AnalyticsDashboard.java`
3. Run: `java AnalyticsDashboard`

## Web Version

### How to Run
1. Open index.html in a web browser
2. Click "Choose File" and upload vgchartz-2024.csv
3. Navigate through the menu options

## Data Format

CSV file should contain:
- Title
- Console
- Genre
- Publisher
- Total Sales
- Release Date
- Last Update
